Simple example of jwt authentication.

## Table of Contents

- Generalinfo

## Introduction

    This small project showcases jwt authentication using acces and refresh token mechanism.

## Technology

    1. NPM
    2. Vanilla javaScript
    3. MongooseJS(MongoDB)
    4. ExpressJS
    5. NodeJS

## Setup

    The app has all the required files and libraries in it in the form of links.
    $ npm install to download required modules and libraries
    $ npm run dev to start backend and frontend in terminal in visual studio code
    some improvements can be made, like better interface.
